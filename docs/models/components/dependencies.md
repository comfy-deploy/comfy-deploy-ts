# Dependencies

The dependencies to use, either as a DepsBody or a list of shorthand strings


## Supported Types

### `string[]`

```typescript
const value: string[] = [
  "Stability-AI/ComfyUI-SAI_API@1793086",
  "cubiq/ComfyUI_IPAdapter_plus@b188a6c",
];
```

### `components.DepsBody`

```typescript
const value: components.DepsBody = {
  dockerCommandSteps: "<value>",
};
```
