# Inputs


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 947371;
```

### `number`

```typescript
const value: number = 7308.56;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `any[]`

```typescript
const value: any[] = [
  "<value>",
];
```
